# 电机设零功能任务

## 目标需求

在 `Motor` 模块中新增将电机当前位置设置为单圈回零零点的功能，默认保存到 EMM 驱动器。

## 涉及模块

- `code/Motor.c`
- `code/Motor.h`
- `code/Emm_V5.h`（仅复用已有接口，不修改）

## 实现思路

新增 `Motor_SetZero(Motor_t *motor)`，对空指针直接返回；有效对象调用
`Emm_V5_Origin_Set_O(motor->addr, 1U)`。不触发电机运动，不更新运行模式与目标缓存。

## 执行步骤

1. 为 `Motor_SetZero()` 编写可验证的失败用例或静态检查用例。
2. 在 `Motor.h` 声明函数，并在 `Motor.c` 实现最小封装。
3. 编译 Keil 工程并检查链接结果。
4. 实机调用后重启驱动器，确认零点保持。

## 验证方法

- 工程编译无新增错误。
- 实机调用接口后，驱动器当前位置被定义为零点。
- 驱动器断电重启后，零点配置仍然有效。

## 已确认决定

- `Emm_V5_Origin_Set_O()` 为底层调用。
- 保存标志固定为 `1U`。
- 本任务不触发实际回零运动。

## 待澄清问题

无。
