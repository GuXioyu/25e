# Motor_CompareLastValue 初始零值比较设计

## 行为

`Motor_CompareLastValue(channel, value)` 将通道的历史值初始视为 0。函数先判断 `value` 是否不同于历史值，再将历史值更新为 `value`。不同返回 1，相同返回 0。

## 数据流

首次云台速度为 20 时，函数比较 20 与初始历史值 0，返回 1；`Task_Motor()` 因此置位 `motor_flag[2]`，并在调度周期内调用 `Motor_SetSpeed()`。

## 边界条件

- 通道越界仍返回 0，且不更新任何历史值。
- 首次输入 0 返回 0。
- 同一数值连续输入仅第一次变化时返回 1。

## 验证

以调用序列 `0, 20, 20, -20` 验证返回值依次为 `0, 1, 0, 1`，并构建 MDK 工程。
